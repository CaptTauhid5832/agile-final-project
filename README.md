# agile-final-project

Final project for Agile Planning and Scrum.

This repository contains the Kanban board, user stories, sprint planning artifacts, and backlog management activities completed as part of the course final project.

## Features

- Product catalog management
- Product creation
- Product retrieval
- Product updates
- Product deletion
- Product likes
- Cloud hosting
- Automated deployments

## Project Board

The project uses a Kanban board to manage the product backlog, sprint backlog, and sprint execution workflow.
